制作了QJoyPad的美观图标

我一直使用 T9键盘 进行软件操作，长时间保持一种工作姿势，手指会有些疼！
所以我想到使用游戏手柄来操作软件，改变手指的操作姿势来缓解疲劳。

在Xubuntu系统(GNU/Linux)中可以使用 QJoyPad 映射游戏手柄到键鼠位。
我已经制作了使用QJoyPad的讲解视频： > 重重点击

视频中演示的是老版本的QJoyPad，设置游戏手柄映射的方法与新版一样。

但新版替换图标的方法与之前不同，以下是新版替换图标的方法：

我制作的 QJoyPad 的美观图标，以 CC0 许可发布，大家可以自由使用。
图标下载 QJoyPad Icon DL: GitHub or 微云

2023-04-16 更新：

图标为 QJoyPad 4.3.1 或更高版本：

  • 将你喜欢的一个PNG图像重命名。
    • 选择橙色或白色或黑色的一个PNG图像
    • 将其复制到用户主目录中，并重命名为：qjoypad.png

  • 在终端中分别执行以下两条命令：
    • sudo cp qjoypad.png /usr/share/icons/hicolor/24x24/apps
    • sudo cp qjoypad.png /usr/share/icons/hicolor/64x64/apps

完成后系统菜单图标与系统托盘图标都会一起更新。
Xfce系统菜单中的 QJoyPad图标也可以使用「菜单编辑器」 Menulibre 这个程序进行编辑。
