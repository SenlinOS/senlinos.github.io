Ibus输入法托盘EN颜色，USB扩展坞等

Ibus输入法

在GNU/Linux系统中，我一直使用Fcitx中文输入法，不过前段时间在输入中文标点时开始变得混乱，因为我当时急需使用，所以把Fcitx输入法御卸掉，安装了Ibus输入法，以前的Ibus拼音已经不再维护，但现在新的Ibus拼音项目又开始活跃。

我们可以安装 ibus 与 ibus-libpinyin

我是使用双拼，Ibus输入法还挺好用。但Ibus输入法托盘中的图标 EN 是深颜色，在深色的Xfce系统栏中不显眼，我以前使用时也想改EN图标的颜色，但没找到，因为这个图标根本不是图片格式，还好搜到了解决的方法：

改变Ibus输入法托盘EN图标的颜色，在终端输入命令：
gsettings set org.freedesktop.ibus.panel xkb-icon-rgba "#E6E6E6"

参考/Reference：https://bbs.archlinux.org/viewtopic.php?pid=1815252#p1815252

USB扩展坞 与 无线鼠标

GNU/Linux 系统中（我使用的是 Xubuntu）当使用USB扩展坞的时候，开机USB设备会失灵，要重新插拔才可以恢复正常使用。我看到有用户的解决方法是运行个脚本，但我试过现在那些脚本已经不灵了。不过我发现了解决的方法：就是使用无线鼠标，对的～使用无线鼠标就不会出现USB设备开机失灵的问题 :)

建设银行 on GNU/Linux

网络银行一般是不支持在GNU/Linux系统与浏览器中使用的，对于个人来说建设银行是个好选择，去柜台使用手机就可以办理银行卡（不要开通网银盾）， 在GNU/Linux上使用网络银行很容易～登陆使用手机验证码就可以，第一次登录建设银行网络银行时要设置登录密码，登录之后要设置用户名。当然使用建设银行手机APP也很方便。

推荐3个安卓软件

现在安卓手机中的APP需要很多权限，不允许联网就不让使用！我推荐 F-Droid 网站中的3个安卓软件，F-Droid中的APP被叫做安卓中的自由软件，所以不会索取不需要的权限。

  • 图库/Simple Gallery Pro：非常好用的相册软件，可以查看图片与视频，可以编辑图片。地址：On F-Droid

  • 文档查看器/MuPDF mini：支持 PDF，XPS，CBZ，无保护 EPUB 和 FB2文档，这个软件小巧、运行快，重要的是支持在原文件中直接进行图片缩放（有的查看软件竟然要先提取图片，然后才能缩放，产生新的多余文件不说，速度还慢）。地址：On F-Droid

  • 记事本软件/Notepad：简单，基本，无不必要服务的笔记应用。地址：On F-Droid

麦克风杂音

在GNU/Linux系统中使用电容麦克风录音时，会出现“突突突”的杂音，也许你可能会想这是GNU/Linux系统驱动的问题，但事实是这与驱动无关。原因是 Wi-Fi 的信号在干扰麦克风，关闭Wi-Fi即可解决问题。
