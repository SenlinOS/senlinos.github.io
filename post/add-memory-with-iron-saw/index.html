使用铁锯给老笔记本电脑添加内存 ;)

我的老笔记本电脑内存只有4G，使用的系统是 Xubuntu GNU/Linux，运行的非常流畅，但使用GIMP编辑大文件时几乎将4G内存占满 (这时系统会去使用硬盘上的交换分区SWAP)，所以我决定给笔记本电脑增加内存。


1. 新买的4G内存很快就到了，店家包装的非常好～开心 :)


2. 笔记本的底板共有5个螺丝，但左上角的螺丝卡死了，怎样都拧不开，还把螺丝帽拧花了 |||


3. 这个螺丝拧不开，就打不开底板！我没有专业的拆解工具，只找到了一根铁锯条，然后我就以最短的直线距离，如上图中把铝合金底板锯开了～


4. 成功打开了铝合金底板 (之前我一直以为底板是塑料的呢)


5. 把新买的内存安装上了～


6. 用透明胶带把锯开的地方贴上。


7. 内存升级到8G，成功了 :)

我的笔记本电脑主要是用来备份文件或停电应急赶稿时使用。
